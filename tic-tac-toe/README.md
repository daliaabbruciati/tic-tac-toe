# Tic-Tac-Toe Game
The classic tic-tac-toe game made in ReactJS.

## Quick start

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

#### Installation
Install the project dependencies by running 
```bash
yarn install
```

#### Running the Project
```bash
yarn start
```
